# Able Restaurant - Full Stack Application

This repository contains the full-stack application for Able Restaurant, located in East Legon, Accra.

## Repository Structure

- **`able restaurant/`**: The frontend website built with React, Vite, and Swiper.
- **`backend/`**: The backend REST API built with Node.js, Express, and MongoDB (Mongoose).

## Getting Started

### Frontend
To run the frontend website locally:
1. Navigate to the `able restaurant` directory:
   ```bash
   cd "able restaurant"
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```

### Backend
To run the backend API locally:
1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```
